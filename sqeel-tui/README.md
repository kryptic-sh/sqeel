# sqeel-tui

Ratatui TUI front-end for [sqeel](https://sqeel.kryptic.sh). Vim-modal SQL
editing on top of [hjkl](https://hjkl.kryptic.sh) bindings, with LSP-aware
completion and hover. Pre-1.0 — API churns.

See the umbrella crate at <https://github.com/kryptic-sh/sqeel> for the full
client.
